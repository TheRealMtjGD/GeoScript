Directories
========================
Below we have a list of all the directories in this project and what they are used for

* .github: Github settings
* .vscode: Vscode launch and debug settings
* build: Stores all the backend build modules
* components: Stores all the universal components used around the program
* debug: Contains temporary debugging infomation
* interface: Stores the frontend cli code
* libraries: Contains the standard library and other preset modules
* logs: Contains temporary logging infomation
* scripts: Contains external scripts for frontend use
* temp: Stores temporary files for compilation
* tests: Test modules for GeoScript

Cache folders
========================
These special folders are cache folders, there folders make the program run faster

* __pycache__
* build/__pycache__
* interface/__pycache__
* components/__pycache__
