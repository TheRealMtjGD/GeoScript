Documentation on how the compiler works
==========================================

This is a folder for people looking to controbuting to GeoScript in every file there is an explanation on some part of the compiler, below are all the files in this directory and what they explain


compiler.md: Gives a general rundown of how the compiler works and where all its important resources are

frontend.md: Explains how all the frontend components work together

backend.md: Explains how all the backend components work together

ram_interface.md: Explains how the virtual ram works and how values are stored in it

interface.md: Explains how the frontend cli works

directories.md: Shows all the directories in this project and what they are for
