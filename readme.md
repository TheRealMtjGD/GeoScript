GeoScript Programming Language
====================================
Latest stable reliese: 1.0.0

#### As of the latest reliese, this compiler only works on Windows 11 / 10, hopefully we can support MacOS in later versions

What is it?
------------------------------------
GeoScript is a staticly-typed object-oriented programming language with a syntax similar to C++ and JavaScript but with a learning curb similar to Python / Lua
